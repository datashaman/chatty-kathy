# chatty-kathy

Experiments with griptape and gradio.

# installation

```
python -m venv .venv
source .venv/bin/activate
pip install -r requirements-dev.txt
```

# running

```
source .venv/bin/activate
python app.py
```

Navigate to [http://127.0.0.1:7860/](http://127.0.0.1:7860/) in your browser.
